Polynomial Calculator Source code on GitHub

This project constructs a polynomial addition program.

  • The executable consumes two filenames from the command line and prints to stdout the sum of the two polynomials found in those files.
  • These polynomials are stored in the following format: 5 9 8 7 6 5 represents 9*x^4 + 8*x^3 + 7*x^2 + 6*x^1 + 5.
  • There is a single line which begins with an integer that represents the length of the polynomial followed by white-space
  • Then white-space separated coefficients in descending order.

Command-Line Minesweeper View on GitHub

    This is a command-line minesweeper game, here are rules and instructions:
  • This project consumes three arguments width height number_of_mines and generates a Minesweeper board internally.
  • The board is a rectangle with a cover over each square.
  • It displays this board to the user and prompt for a coordinate where they predict that no mine exists.
  • If users struck a mine then they lose (let user know).
  • Otherwise uncover the selected square. If there are no adjacent mines then it should be blank and you uncover all adjacent squares too.
  • Otherwise show the number of mines in the surrounding 8 squares.
  • If at somepoint only mines are covered then the user wins and the game ends.

Gee-Mail View on GitHub

This project is a secure asynchronous (same-machine) message platform. Here is what it can do:

  • When loaded, prompt the user to sign-in or register.
  • When a user registers, get their username and password.
  • When a user logs in, ask for username and validate password. If login fails, choose your response.
  • Once logged in, tell them the number of messages they have, list the users that have sent them a message. Then ask if they want to read messages or write messages.
  • When writing messages, prompt them for the receipient username, the message, and a shared passphrase with that receipient.
  • When they read a message, let them select the message somehow, then prompt them for the shared passphrase with that receipient, then show a plaintext message.
